SimpleImputer – Quick Notes

Definition:
SimpleImputer is a preprocessing tool used to handle missing values (NaN) in a dataset by replacing them with simple statistical values.

Why use it:
Most machine learning models cannot handle missing data. SimpleImputer helps clean the dataset before training.

Strategies:

* mean → replaces missing values with average (numeric)
* median → replaces with middle value (numeric, good for outliers)
* most_frequent → replaces with mode (categorical)
* constant → replaces with a fixed value (e.g., 0 or "unknown")

When to use:

* Use mean → when data is clean and no major outliers
* Use median → when data has outliers
* Use most_frequent → for categorical features
* Use constant → when missing values have meaning

Example:
Age: [25, 30, NaN, 35]
→ median = 30 → missing value becomes 30

Key points:

* Apply before model training
* Handle numeric and categorical columns separately
* Common step in ML pipelines
* Prevents errors due to missing values
