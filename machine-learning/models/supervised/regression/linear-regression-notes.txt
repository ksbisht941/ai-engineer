Linear Regression – Quick Notes

Definition:
Linear Regression is a supervised learning algorithm used to model the relationship between a dependent variable (target) and one or more independent variables (features).

---

1. Simple Linear Regression

Definition:
Models relationship between one feature and target.

Mathematical Expression:
y = β0 + β1x + ε

Best Fit Line (Closed-form solution):
β1 = Σ(xi − x̄)(yi − ȳ) / Σ(xi − x̄)²
β0 = ȳ − β1x̄

Where:

* y → target variable
* x → feature
* β0 → intercept
* β1 → slope (coefficient)
* ε → error term
* x̄ → mean of x
* ȳ → mean of y

Use case:

* Predicting salary based on experience

---

2. Multiple Linear Regression

Definition:
Models relationship between multiple features and target.

Mathematical Expression:
y = β0 + β1x1 + β2x2 + ... + βnxn + ε

Matrix Form (Best fit):
β = (XᵀX)⁻¹Xᵀy

Where:

* x1, x2, ..., xn → multiple features
* β1, β2, ..., βn → coefficients

Use case:

* Predicting house price using area, rooms, location

---

3. Polynomial Regression

Definition:
Extends linear regression to model non-linear relationships using polynomial terms.

Mathematical Expression:
y = β0 + β1x + β2x² + β3x³ + ... + βnxⁿ + ε

Use case:

* When data shows curved relationship (not straight line)

---

Simple Example:

Dataset:
x → [1, 2, 3]
y → [2, 4, 5]

Step 1: Mean
x̄ = 2
ȳ = 3.67

Step 2: Compute β1
β1 = 1.5

Step 3: Compute β0
β0 = 0.67

Final Model:
y = 0.67 + 1.5x

Prediction:
x = 4 → y = 6.67

---

Key Points:

* Still called “linear” because it’s linear in coefficients (β)
* Assumes linear relationship between features and target
* Sensitive to outliers
* Requires feature scaling (especially for polynomial)
* Polynomial can overfit if degree is too high
* Best fit line minimizes RSS (error)
* Closed-form solution works for small datasets

---

Shortcut:

* Simple → 1 feature
* Multiple → many features
* Polynomial → handle non-linearity
* Best fit → minimize error (RSS)
